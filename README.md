# rus_backtest: Trading Strategy Backtesting Engine (Go)

Система для бэктестинга торговых стратегий на языке Go. Проект реализует полноценный цикл симуляции торговли на исторических данных с использованием технического анализа и риск-менеджмента.

## 🚀 Основные возможности
- **Технические индикаторы**: Собственная реализация EMA, RSI и ATR.
- **Детекция уровней**: Автоматическое определение горизонтальных уровней поддержки/сопротивления и зон консолидации.
- **Анализ трендов**: Алгоритм поиска локальных минимумов/максимумов и построение трендовых линий.
- **Торговый движок**: Поддержка стратегий на отскок (Bounce) и пробой (Breakout) с учетом комиссий и проскальзываний.
- **Риск-менеджмент**: Динамический расчет лота, лимиты на дневную прибыль/убыток и стоп-лоссы по ATR.

## 🛠 Технологии
- **Language**: Go (Golang)
- **Architecture**: Modular (pkg/internal split)
- **Data Format**: JSON (Open/High/Low/Close/Volume)

## 📊 Стратегия "Снайпер v2" (Конфигурация по умолчанию)
В текущей версии настроена стратегия со следующими параметрами:
- **Риск на сделку**: 1.8%
- **RR Ratio**: 1 : 4.0
- **Макс. сделок в день**: 3
- **Фильтрация**: Учет корреляции с BTC (порог > 0.92)

## 📦 Структура проекта
- `pkg/indicators` — математика индикаторов.
- `internal/analysis` — логика поиска паттернов на графике.
- `internal/app/trading` — симуляция исполнения ордеров.
- `internal/metrics` — расчет Profit Factor, Drawdown, Sharpe & Sortino ratios.
